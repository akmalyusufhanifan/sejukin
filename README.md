Getting Started

1. Clone Repository
   git clone https://github.com/yourusername/sejukin.git
2. Navigate to Project Folder
   cd sejukin
3. Install Dependencies
   npm install
4. Run Development Server
   npm run dev

Open:

http://localhost:3000
Production Build

To create an optimized production build:

npm run build

To run production mode locally:

npm run start
